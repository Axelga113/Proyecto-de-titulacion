# Proyecto de Titulación — Mitigación de Ataques DDoS en Redes SDN

Este repositorio contiene el código desarrollado para el proyecto de titulación enfocado en la evaluación y comparación del comportamiento de redes convencionales y redes SDN frente a ataques DDoS. 

El trabajo implementa simulaciones en Mininet y el controlador Ryu, incorporando distintas estrategias de mitigación, las cuales son:
- detección por umbral (THRESHOLD).
- limitación de tasa (RATE LIMITING).
- bloqueo selectivo (BAN).

El repositorio está organizado en dos componentes principales:
- Red Convencional: scripts de topología, generación de tráfico y medición de rendimiento sin plano de control SDN.
- Red SDN y Mitigaciones: topologías SDN, controladores y mecanismos de mitigación.

Documentación complementaria y explicaciones detalladas del funcionamiento interno se encuentran en los archivos README específicos dentro de cada carpeta.
